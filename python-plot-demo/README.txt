Plotting using Python (matplotlib library)

(1) Download and install anaconda for Python 3 from www.anaconda.com

(2) From Terminal, type "jupyter notebook" then enter, a browser window will pop up 

(3) Navigate to the folder with the Jupyter notebook *.ipynb file and data file *.csv

(4) Click *.ipynb file, and start to run each cell one by one from top to bottom by clicking "run" button or press shift+enter key combination. 

(5) Modify the python code to fit your need, like changing the columns to be plotted, etc.

Enjoy!
